West Texas data center build-out map

Includes proposed / announced / under construction / operating data center sites within roughly a 75-mile radius of Abilene, based on sources reviewed as of 2026-02-28. Overlapping entries were consolidated so multi-building campuses appear as a single site marker unless there is a distinct separate project. Some coordinates remain approximate where public records gave a street address or tract description but not a parcel-accurate map point.
